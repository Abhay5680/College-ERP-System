# 🎓 College ERP System

A modern **College ERP (Enterprise Resource Planning) System** built using **Java Servlets, JSP, JDBC, MySQL, and Bootstrap 5** following the **MVC Architecture**. The system provides a responsive admin interface to manage academic information efficiently.

---

## 📌 Features

- 🔐 Secure Admin Login
- 👨‍🎓 Student Management
- 👨‍🏫 Faculty Management
- 🏢 Department Management
- 📚 Course Management
- 📖 Subject Management
- 👤 Admin Profile Management
- 📱 Responsive Bootstrap UI
- 🏗️ MVC Architecture
- 🗄️ MySQL Database Integration

## 🚀 Getting Started

### Prerequisites

- Java JDK 8+
- Eclipse IDE
- Apache Tomcat 9+
- MySQL Server

### Installation

1. Clone the repository

```bash
git clone https://github.com/Abhay5680/College-ERP-System.git
```

2. Import the project into Eclipse.

3. Create a MySQL database.

4. Update your database credentials in the JDBC connection file.

5. Add Apache Tomcat to Eclipse.

6. Run the project.

7. Open:

```
http://localhost:8080/CollegeERP/
```
